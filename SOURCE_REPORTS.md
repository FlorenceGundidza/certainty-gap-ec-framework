## Retrieval pointers
- **ADB** (28): Independent Evaluation document repository (PCR / PVR / TCR).  https://www.adb.org/projects/documents
- **AfDB** (17): Project appraisal & completion reports.  https://www.afdb.org/en/documents
- **JICA** (5): Ex-post evaluation reports.  https://www2.jica.go.jp/en/evaluation/
- **Other** (5): World Bank / other IFI document portals.

All documents are public-sector evaluation reports. We release only the *derived* data (extracted facts, benchmark items, labels, model outputs) under CC BY 4.0; reuse of the original PDFs is governed by each issuer's terms.
